Common HTML

These are the most commonly used HTML tagsand a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page

body

Defines the document body

div

Defines a section

main

Specifies the main content of a document

footer

Defines footer in a document or section

li

Defines a list term

head

Defines info about a document

section

Defines a section

article

Defines an article

p

Defines a paragraph

html

Contains whatever is in it in a html format

header

Defines text as a header

nav

Defines navigation links

aside

Defines content aside from the page

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink

strong

Bolds text

em

Emphasizes text

img

Defines image

span

Defines a section
